# 數學冒險樂園（math-fun-kids）

這是一個專為**國小 1-3 年級**設計的網頁數學遊戲。  
不需要安裝任何套件，**直接開啟 `index.html` 就可以玩**。

---

## 🎮 遊戲內容

### 1) 加減法闖關（循序漸進）

依照孩子答對題數自動升級，從簡單到進階：

- **等級 1**：1 位數加減法
- **等級 2**：2 位數加減法
- **等級 3**：3 位數加減法

> 減法題目會避免出現負數答案，降低學習挫折感。

### 2) 九九乘法練習

- 題目範圍為 `1 × 1` 到 `9 × 9`
- 可搭配連續答對與分數機制，增加挑戰與成就感

---

## ✨ 功能特色

- 兩種模式一鍵切換（加減法 / 九九乘法）
- 分數、等級、連續答對即時顯示
- 連續答對可獲得星星獎勵（每 3 題 1 顆）
- 錯題立即顯示正確答案
- 支援按 Enter 快速送出答案
- 內建遊戲音效（答對、答錯、升級）且可一鍵開關
- 加減法升級時會出現闖關動畫提示
- 介面使用大按鈕與清楚字體，適合低年級學童
- 程式碼內含詳細**正體中文註解**

---

## 🚀 如何使用

1. 下載或複製這個專案
2. 用瀏覽器直接開啟 `index.html`
3. 點擊「開始遊戲」即可開始作答

---

---

## 🧠 遊戲規則（簡述）

- 答對：+10 分，連續答對 +1
- 每連續答對 3 題：+1 顆星星 ⭐
- 答錯：-2 分（最低不低於 0），連續答對歸零
- 可隨時切換模式或重設進度
- 可自行切換音效開/關

## 🛠 GitHub Pages（可選）

如果你要讓別人直接用網址玩：

1. 到 GitHub 專案頁面點選 **Settings**
2. 進入 **Pages**
3. Source 選擇目前分支（例如 `main`）與根目錄（`/root`）
4. 儲存後等待部署完成
5. 用 GitHub 提供的網址即可直接遊玩
